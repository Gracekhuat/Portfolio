## About

This portfolio was developed to present my technical skills, academic projects and professional background for internship opportunities in **Data Analytics**, **Business Intelligence** and **Information Technology**.

The website highlights my experience in data analysis, dashboard development, web development and database systems through a clean, responsive and user-friendly design.


## Features

- Responsive design for desktop, tablet and mobile devices
- Modern and clean user interface
- About Me and Technical Skills sections
- Featured Projects showcase
- Education timeline
- Downloadable resume
- Contact form powered by Formspree
- Smooth scrolling and fade-in animations

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Bootstrap Icons
- Google Fonts
- Formspree
